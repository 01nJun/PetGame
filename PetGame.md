# 🐾 Pet Game (펫 키우기 게임) 개발 일지

---

## 1일차: 프로젝트 기획 및 기본 구조 설정

### 프로젝트 주제
- **주제:** Pet Game (펫 키우기)
- **육성 펫:** Duck(오리), Hamster(햄스터)

### 기능을 정하고 기능별 획득량 조정 
- **밥 주기:** 배고픔 10 감소, 행복도 5 감소
- **놀아주기:** 배고픔 5 증가, 행복도 5 증가
- **산책하기:** 배고픔 5 증가, 행복도 5 감소
- **잠재우기/깨우기:** 능력치 변화 없음

---

## 2일차: 이벤트 로직 구상 및 구현

###  산책 이벤트 
'산책하기'를 눌렀을 때 확률적으로 이벤트가 발생하도록 구현했습니다.

| 이벤트 번호 | 내용 | 행복도 | 배고픔 | 경험치 |
| :--- | :--- | :---: | :---: | :---: |
| 0 | 친구 만나기 | ▲ +15 | ▲ +5 | ▲ +10 |
| 1 | 간식 줍기 | ▲ +5 | ▼ -10 | ▲ +5 |
| 2 | 소나기 만나기 | ▼ -10 | ▲ +5 | - |
| 3 | 아무일도 X | ▼ -5 | ▲ +5 | - |

### 놀아주기 이벤트 추가
펫의 행복도가 90 이상일 때 '놀아주기'를 하면 20% 확률로 펫이 선물을 가져옵니다.
- **효과:** 배고픔 10 감소

| 이벤트 번호 | 내용 | 행복도 | 배고픔 | 경험치 |
| :--- | :--- | :---: | :---: | :---: |
| 0 | 아무것도 X | - | ▲ +5 | - |
| 1 | 아무것도 X | - | ▲ +5 | - |
| 2 | 아무것도 X | - | ▲ +5 | - |
| 3 | 아무것도 X | - | ▲ +5 | - |
| 4 | 놀아주기 이벤트 | - | ▼ -10 | - |

### '피로도' 시스템 추가 및 재설계
펫의 피로도를 추가해 플레이어가 펫의 피로도를 관리하도록 '재우기/깨우기' 기능의 중요성을 높였습니다.

- **재우기:**  재우기 상태에서는 다른 행동(밥주기, 놀아주기, 산책하기)이 불가능합니다.
- **자는 중 행동 시도:** "자는 중이라 안됩니다." 라는 알림창이 뜨며, 창을 닫으면 피로도가 감소합니다.
- **깨우기:**  깨우기 버튼을 누르면 펫이 다시 활동 상태가 됩니다.
